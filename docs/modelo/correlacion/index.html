Correlación entre bases de datos

Correlación entre bases de datos in situ y satelitales.

En la siguiente sección se presenta la correlación entre cada una de las bases de datos satelitales y la información in situ correspondiente a las variables de interés seleccionadas.

Radiación Solar#

Para la variable de radiación solar la base de datos satelital que tiene el valor de correlación más grande con las mediciones in situ es SOLCAST con $0.90$. NSRDB también cuenta con un valor alto de $0.85$.

Por otra parte ERA5 tiene una correlación media, acorde a su documentación se debe tener cuidado al comparar el dato con mediciones in situ ya que la variable equivalente a la medición de un piranómetro es modelada como un punto concreto del espacio y el tiempo, en lugar de representar el promedio de una cuadricula del modelo y un intervalo de tiempo correspondiente.


Fig. 1 Mapa de calor de Correlación para la Radiación Solar entre bases de datos satelitales y mediciones in situ.


Fig. 2 Diagramas de dispersión de Radiación Solar entre bases de datos satelitales y mediciones in situ.

Velocidad del Viento#

La correlación más alta con las mediciones in situ se da en la base de datos de NSRDB con un valor de $0.78$. Mientras que las otras bases de datos tienen un valor de correlación medio.


Fig. 3 Mapa de calor de Correlación para la Velocidad del Viento entre bases de datos satelitales y mediciones in situ.


Fig. 4 Diagramas de dispersión de Velocidad del Viento entre bases de datos satelitales y mediciones in situ.

Humedad Relativa#

En este caso la base de datos de ERA5-Land es la que presenta el mayor valor de $0.79$, mientras que NSRDB y SOLCAST tienen un valor de correlación medio.


Fig. 5 Mapa de calor de Correlación para la Humedad Relativa entre bases de datos satelitales y mediciones in situ.


Fig. 6 Diagramas de dispersión de Humedad Relativa entre bases de datos satelitales y mediciones in situ.

Temperatura#

Al igual que con la Humedad Relativa ERA5-Land tiene la correlación más alta respecto a las demás bases de datos, con un valor de 0.86. Esto se da debido a su gran resolución y su enfoque de optimizar las medidas en tierra.
