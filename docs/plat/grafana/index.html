Visualización de Datos

Visualización de Datos.

Grafana es una aplicación de código abierto que facilita el desarrollo de dashboards sobre datos de series de tiempo en grandes volúmenes y en tiempo real. Brinda la posibilidad de visualizar, consultar, establecer alertas y obtener información de métricas de infraestructuras (servidores, routers, etc.), aplicaciones propias y múltiples tipos de bases de datos, como InfluxDB [].
